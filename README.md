# MBv4 Program Builder

A block-based programming system that models program structure as an AST, renders blocks visually, supports nesting, validates structure, and generates code.

## Tech Stack

- React 18
- TypeScript (strict)
- Vite

## Getting Started

```bash
npm install
npm run dev
```

## Project Structure

```
src/
  components/   # React UI components
  core/         # Business logic (block factory, validator, codegen)
  hooks/        # Custom React hooks
  utils/        # Utility functions
  types/        # TypeScript type definitions
docs/           # Documentation
tests/          # Tests
```
